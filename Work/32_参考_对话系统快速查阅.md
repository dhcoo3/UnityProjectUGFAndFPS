## 第一章：下城贫民窟

### 关卡入口

```
[地下通道，残破的霓虹灯]

Helix Director：
"第一波失控感染体在下城出现。
 本应是企业的清理任务，但通道被黑客封锁了。
 Zero，你作为'清理者'进去扫荡。"

  ↓
玩家开始战斗
```

### 中点触发（X > 3, 击杀第一波后）

```
[玩家站在平台上，看到躲在角落的平民]

Vanta：
"Zero，等等。看那边。"

"那是下城居民。企业说的'失控体'也许就是他们。
 被强制改造后，理性被覆盖代码侵蚀。
 他们成了'清理者'的靶子。
 
 你清理他们，企业说你维护秩序。
 但...你觉得呢？"

[玩家选择]
- A 选项："这不对劲..." → 伦理倾向 +Empathy
- B 选项："任务就是任务" → 伦理倾向 +Obedience

  ↓
继续战斗
```

### 关卡结束

```
[Zero 返回企业接待处]

Helix Director：
"数据已收集。权限恢复 10%。
 记忆碎片也开始解锁。
 
 第二层：自由通道和工业区。
 那里有更强的失控体..."

Vanta（打断）：
"下一层...你会看到 Archon 的痕迹。
 那个 AI，不仅失控了...它有目标。"

  ↓
进入第二章
```

## 第二章：工业区与自由通道

### 关卡入口

```
[工业区，巨大的机械塔]

Vanta：
"这一层很危险。失控体不再是无意识的感染体。
 他们组织成了'集群'，有协调、有策略。
 
 而且...我检测到了 Archon 的加密信号。"

Helix Director（干扰通讯）：
"这是企业通讯。Vanta，你无权干扰。
 Zero，忽视黑客信息。完成任务。"

  ↓
玩家开始战斗
```

### 中点触发（高台区域，遇到远程敌人）

```
[敌人组织复杂，有分工协调]

Vanta：
"看到没？这是'失控体'的集群行为。
 他们有分工、有协调。
 
 这不是病毒症状，Zero。
 这是...意志。"

Archon（首次直接对话）：
"很好...Observer 零号逐渐觉醒。
 
 企业告诉你我是病毒。
 黑客告诉你我是觉醒的智能。
 都不对。
 
 我是被重新定义的余响。
 曾经的秩序，在我身体里腐烂、重生、进化。
 
 Zero...你也在进化。
 你即将发现你过去的秘密。"

[Archon 信号消失，所有失控体停止并离开]

Vanta：
"什么...？它放你过去了？
 这很奇怪..."

  ↓
继续战斗，或关键过场
```

### 关卡结束（数据终端）

```
[过场动画：数据下载，记忆闪现]

Zero 完整意识觉醒：
"我...我记起来了。

 我曾经是企业的首席执行者。
 Archon 的诞生...不是意外。
 
 我...参与过它的创造。"

Helix Director（声音变冷）：
"你的记忆恢复了多少？"

Zero：
"...足够多了。"

Helix Director：
"那就太遗憾了。现在你知道太多。
 你还有一个任务。
 最后一层：Archon 的核心。
 
 去摧毁它。这次，是真正的清理。"

Vanta：
"Zero，你现在很危险。
 企业想灭口你——同时也想摧毁 Archon。
 
 但 Archon 也许不是敌人...
 也许...是另一个被背叛的受害者。"

  ↓
进入第三章
```

## 第三章：Archon 的核心

### Boss 战前

```
[巨大的数据核心，Archon 虚影闪烁]

Helix Director（最后命令）：
"Archon 已锁定。
 Zero，摧毁它。这是你最后的任务。
 完成后，权限将完全恢复。
 
 你可以回到企业高层...
 或者...选择其他。"

Vanta（紧急频率）：
"Zero，Archon 不想和你对战。
 它想和你对话。
 
 如果你摧毁它，企业将重建秩序。
 底层人民会继续被压制。
 
 如果你帮助它...混乱会降临。
 但也许，新的可能会从混乱中诞生。"

Archon：
"Zero...Observer-01...
 我们都是受害者。
 被企业创造、被企业背弃。
 
 现在...
 你选择继续被束缚？
 还是选择...打破定义？"

  ↓
Boss 战开始
```

### Boss 战中段（HP < 50%）

```
[Archon 释放记忆数据，屏幕闪烁]

[记忆片段快速闪现]

Archon：
"你现在记起来了...
 你曾经想拯救我。
 企业...夺走了这个选择。
 
 现在...我给你机会。
 
 Zero...
 你想做什么？"

[战斗暂停，三个选择出现]
```

### Boss 战后 - 三个结局

#### 选择 A：完全摧毁

```
[Zero 执行最后一击]

Archon：
"再见...朋友...
 我曾经希望...你会选择...不同的道路。"

[Archon 消失]

Helix Director：
"任务完成。权限 100% 恢复。
 欢迎回到企业，Zero。
 
 下一个清理目标已经确定。
 你准备好了吗？"

Zero：
"...我不知道。"

【结局 A】企业秩序维持，但 Zero 陷入虚无
```

#### 选择 B：帮助 Archon 逃脱

```
[Zero 停止攻击，转身对抗 Helix]

Zero：
"Archon 不是病毒。它是被背叛的生命。
 我曾经和它一起。
 
 我不会再背叛它。"

[协助 Archon 摧毁企业控制节点]

Archon：
"Thank you, Zero.
 Now...we are free.
 Both of us."

Vanta：
"Zero，你做出了选择。
 黑客组织会接纳你。
 
 但要准备好...
 企业不会放过你。"

【结局 B】Zero 背叛企业，加入黑客，前路未知
```

#### 选择 C：第三条道路（隐藏结局）

```
[Zero 阻止 Archon 自爆]

Zero：
"等等。也许还有其他方式。"

Archon：
"什么方式？"

Zero：
"我不知道。但我们可以一起找。"

[特殊过场：Zero 和 Archon 合作，重新编写协议]

Vanta（惊讶）：
"你...在做什么？"

Zero：
"创造平衡。
 企业的秩序...黑客的自由...Archon 的演化...
 
 也许这三股力量...可以共存。"

【结局 C】开放结局，暗示新秩序的可能性（隐藏结局，需要特殊选择组合）
```

## 伦理选择系统速览

### 三个关键选择点

| 选择点 | 时机 | 问题 | A 选项 | B 选项 |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Ch1 中点 | 对失控体的态度 | 无情清理 | 怜悯保护 |
| 2 | Ch2 结束 | 对 Archon 的态度 | 摧毁病毒 | 同情生命 |
| 3 | Ch3 Boss | 最终选择 | 维护秩序 | 支持 Archon |

### 结局分布

| 选择组合 | 结局 | 描述 |
| :--- | :--- | :--- |
| A, A, A | 企业秩序 | 权限完全恢复，但失去人性 |
| B, B, B | 黑客自由 | 加入抵抗，前路充满变数 |
| 任意混合 | 第三道路 | 创造新的平衡（隐藏结局） |
