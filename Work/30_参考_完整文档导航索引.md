## 📋 三章开发任务总览

### 第一章：基础闭环章（登录 → 进图 → 战斗 → 结算）

**地图参数**：
- 场景名：`ChapterO1_MapO1`
- 尺寸：30 × 8 Tile
- 玩家出生：(0, 1) → 结束点：(29, 1)
- 特点：直线推进，3 波怪，无复杂机制

**关键任务**：
1. ✅ MCP 创建地图（根据矩阵）
2. ✅ 导出碰撞数据
3. ✅ 配置表填写（MapData, MainChapter, MonsterSpawner, Monster, AI）
4. ✅ 代码对接（FightProxy, MapProxy, MonsterSpawnerProxy）
5. ✅ 验证通关流程

**验收标准**：能稳定进入 → 玩家可操作 → 怪物按时刷新 → 完整结算

📖 详细内容：见 **33_参考_AI开发路线图和任务清单.md** 第 1 章

---

### 第二章：技能与空间章（高低差 → 远程怪 → 平台 → AOE）

**地图参数**：
- 场景名：`ChapterO2_MapO2`
- 尺寸：35 × 12 Tile
- 玩家出生：(0, 3) → 结束点：(34, 1)
- 特点：高低差、移动平台、远程火力、技能拆场

**关键任务**：
1. ✅ MCP 创建地图（包含高台和移动平台）
2. ✅ 配置移动平台运动（PlatformDef）
3. ✅ 配置远程怪和 AI
4. ✅ 配置 AOE 技能和 Buff
5. ✅ 验证高低差、平台、远程火力、技能拆场

**验收标准**：平台正常 → 远程怪有压制 → 技能能拆场 → 无卡死

📖 详细内容：见 **33_参考_AI开发路线图和任务清单.md** 第 2 章

### 第三章：Boss 与结算章（Boss 多技能 → 结算 → 完整流程）

**地图参数**：
- 场景名：`ChapterO3_MapO3_Boss`
- 尺寸：40 × 10 Tile
- 玩家出生：(0, 2)，Boss 出生：(19, 2)
- 特点：宽敞竞技场，Boss 多技能多阶段

**关键任务**：
1. ✅ MCP 创建 Boss 竞技场

5. ✅ 验证完整 Boss 战和结算

**验收标准**：Boss 战正常 → 多技能释放 → 胜负结算正确 → 能返回主菜单

## 🐛 常见问题排查

| 问题 | 原因 | 解决方案 |
| :--- | :--- | :--- |
| 地图碰撞无法加载 | 导出路径错误或文件名不匹配 | 检查 MapData.Name 是否与导出文件名一致 |
| 怪物刷不出来 | TriggerX 条件未满足或 MonsterId 无效 | 检查 MonsterSpawner.TriggerX 和 MonsterId 是否正确 |
| 技能无法释放 | SkillDef 或 TimelineId 无效 | 检查 SkillDef.TimelineId 是否指向有效的 TimeLineDef |
| Boss 卡住不动 | AI 条件过严格或陷入碰撞 | 检查 AIActionDef 的 Conditions，放宽距离限制 |

详细排查指南：见 **07_AI_Development_Roadmap.md** 第 5 章

## 📌 重要提示

- ⚠️ **所有坐标使用 Tile 单位**（不是像素）
- ⚠️ **所有数值使用定点数 fix 类型**（不是 float）
- ⚠️ **导表后一定要检查无 ERROR**
- ⚠️ **新增 UI 或数据结构要同步更新这些文档**
