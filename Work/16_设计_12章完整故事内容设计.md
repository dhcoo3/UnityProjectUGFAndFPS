## 2. 地图设计系统升级

### 新的地图符号标准

```
垂直元素的新符号：

基础符号（同前）：
0 = 空气
1 = 地面（固定）
2 = 单向平台（下方可穿）
3 = 移动平台
P = 玩家出生点
M = 怪物刷新
E = 关卡结束点

新增符号：
4 = 破碎平台（不稳定，需要快速通过）
5 = 冰冻平台（滑动）
6 = 反弹垫（跳得更高）
7 = 传送门（快速传送）
8 = 激光陷阱（伤害）
9 = 移动墙壁（动态阻挡）
# = 上升气流（增加跳跃高度）
= = 下降气流（减少跳跃高度）
@ = 磁性平台（吸附玩家）
$ = 能量补给点
```

### 新地图设计原则

1. **高度利用**：地图高度 15-25 Tile
2. **平台密度**：每层有 2-4 个跳跃路径
3. **垂直分层**：
   - 底层（0-5）：主战斗区
   - 中层（5-15）：躲避区和探索区
   - 顶层（15-25）：Boss 或关键事件

4. **难度设计**：
   - 简单路径（直接通过）
   - 困难路径（更多奖励）
   - 隐藏路径（额外内容）
