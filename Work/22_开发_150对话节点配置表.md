## 对话配置表说明

| 字段 | 说明 |
| :--- | :--- |
| Id | 对话唯一 ID（递增） |
| Chapter | 所属章节（Intro, 1-12） |
| NodeId | 节点标识符（便于代码引用） |
| Speaker | 说话人（Helix, Vanta, Archon, Zero, SystemNarrator 等） |
| Text | 对话文本（支持换行） |
| Type | 类型（Auto 自动播放, Choice 需要选择, Cutscene 过场） |
| Duration | 自动播放时长（秒，仅 Auto 类型） |
| NextNodeId | 下一个对话 ID（仅 Auto 类型） |
| ChoiceIds | 选择对话的分支 ID，逗号分隔（仅 Choice 类型） |
| ChoiceTexts | 选择按钮文字，逗号分隔（仅 Choice 类型） |
| TriggerCondition | 触发条件（如 X>3, Wave1Killed 等） |
| VoiceId | 语音资源 ID（0 表示无语音） |
