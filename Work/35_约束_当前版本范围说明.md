# 当前版本范围说明

本文档用于固定自动开发的当前边界，避免把扩展内容误当作本轮主线。

## 1. 当前主线

本轮只做三章闭环：

1. 登录
2. 进图
3. 基础战斗
4. 结算退出

对应工程目标：

- `PreloadProcedure -> ChangeSceneProcedure -> LoginProcedure -> RunGameProcedure`
- `FightProxy`
- `MapProxy`
- `UnitProxy`
- `MonsterSpawnerProxy`
- `SkillProxy`
- `TimelineProxy`
- `DamageProxy`
- `AIProxy`
- `GameOverProcedure`

## 2. 本轮必做内容

- 跑通登录到主线关卡的完整链路。
- 跑通普通攻击、技能、伤害、刷怪、死亡、结算。
- 跑通章节入口和战斗 HUD。
- 跑通胜利、失败、返回主菜单、重新挑战。
- 跑通三章主线内容落地。

## 3. 本轮不做内容

以下内容不作为当前自动开发主线：

- 12 章完整扩展版
- 大规模 Meta 系统
- 商店、背包、存档、成就
- 本地化全量替换
- 大重构或重写现有框架
- 额外平台化、运营化能力

## 4. 文档优先级

优先级从高到低：

1. 当前版本范围说明
2. AI 开发交接说明
3. 三章路线图
4. 配置与地图专项文档
5. 12 章扩展参考

## 5. 统一口径

- `MapData.Name` 代表场景或资源名。
- `MainChapter.Name` 代表章节展示名。
- `3` 章闭环是主线基线。
- `12` 章内容仅作为后续扩展参考。
- 若与当前代码冲突，以当前代码仓库真实状态为准。
