# 3. UI/UX 交互说明文档 (UI/UX Spec)

## 3.1 UI 核心流程
1. **加载页 (Splash/Loading)**：资源预加载。
2. **主界面 (Main Menu)**：包含出战、背包、强化、设置。
3. **战斗 HUD**：
    - 左上：生命条 (HP)、技能槽 (SP)。
    - 右下：当前武器、余弹、换弹进度条。
    - 中间/角色头顶：伤害飘字、负面状态。
4. **结算页 (Result)**：奖励展示、评分、返回。

## 3.2 交互反馈规范
- **按键反馈**：点击瞬间缩放 (0.95x)、播放电子音效。
- **状态提示**：
    - 换弹中：枪械图标半透明，进度环循环。
    - 技能冷却中：遮罩倒计时。
- **伤害飘字**：
    - 玩家受伤：红色，向左/右跳动。
    - 敌人受伤：白色/黄色(暴击)，向上漂浮并逐渐消失。

## 3.3 分辨率与适配
- **参考分辨率**：1920x1080 (16:9)。
- **适配方案**：
    - 核心元素居中对齐或边角锚定。
    - 界面背景支持 9-Slice 或 自动拉伸。

## 3.4 字体与色彩 (赛博风格)
- **主色调**：荧光蓝 (#00F0FF)、赛博黄 (#F0E68C)、霓虹紫 (#FF00FF)。
- **字体**：等宽字体或具有科技感的非衬线字体。
