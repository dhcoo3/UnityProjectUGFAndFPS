## 1. 矩阵符号参考

| 符号 | 名称 | 含义 | 碰撞 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| `0` | 空气 | 无碰撞区域 | 无 | 玩家可通过 |
| `1` | 地面 | 固定阻挡 | 有 | 玩家/怪物可站立 |
| `2` | 单向平台 | 从下可穿，上可站 | 单向 | 用于跳跃设计 |
| `3` | 移动平台 | 会运动的平台 | 有 | 需在 PlatformDef 配置 |
| `P` | 玩家出生 | 玩家初始位置 | 有 | 该位置也是地面 |
| `M` | 怪物刷新 | 怪物出现点 | 无 | 仅用于配置参考 |
| `E` | 关卡结束 | 胜利触发点 | 无 | 进入即结算 |
| `B` | Boss 出生 | Boss 初始位置 | 无 | 等同于 M，用于 Boss 标记 |

## 2. 坐标系转换

### 矩阵坐标 → 游戏坐标

```
游戏坐标 = 矩阵坐标 × Tile_Size + Offset

其中：
- Tile_Size：1 Tile = 1 游戏单位（可根据项目调整）
- Offset：世界坐标偏移（一般为 0）

例：
- 矩阵 (0, 1) → 游戏 (0, 1)
- 矩阵 (9, 6) → 游戏 (9, 6)
```

### 矩阵中的行列顺序

```
矩阵按以下方式排列：
- 行：从上到下，Y 轴递减
- 列：从左到右，X 轴递增

示例：
第一行（Y最高）
第二行
...
最后一行（Y最低）
^
X 轴从左到右 →
```

## 8. 配置表文件清单

所有需要填写的 Excel 文件：

```
Config/DataTables/Datas/

├── Map/
│   ├── #MapData.xlsx           （需改：加 3 个地图）
│   └── #PlatformDef.xlsx       （需改：加 1 个移动平台）
├── Spawner/
│   └── #MonsterSpawner.xlsx    （需改：加 8 个刷怪点）
├── Monster/
│   └── #Monster.xlsx           （需改：加 4 种怪物）
├── AI/
│   ├── #AIGroupDef.xlsx        （需改：加 4 个 AI 组）
│   └── #AIActionDef.xlsx       （需改：加 16 个 AI 行为）
├── Skill/
│   ├── #SkillDef.xlsx          （需改：加 5 个技能）
│   ├── #TimeLineDef.xlsx       （需改：加 5 个时间线）
│   ├── #BulletDef.xlsx         （需改：加 2 种子弹）
│   ├── #SkillBulletDef.xlsx    （按需）
│   ├── #SkillAoeDef.xlsx       （需改：加 1 个 AOE）
│   ├── #AoeDef.xlsx            （需改：加 1 个 AOE）
│   └── #BuffDef.xlsx           （按需）
└── MainChapter/
    └── #MainChapter.xlsx       （需改：加 3 个章节）
```
